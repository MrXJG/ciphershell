# CipherShell v0.1.0-rc2

Release type: internal release candidate.

## Scope

- Product rename from GMSSH Client to CipherShell for user-facing app, installer, package, shortcut, and release artifact names.
- Internal compatibility identifiers remain stable where they protect existing saved profiles, credentials, GM engine environment variables, and regression tooling.
- macOS arm64 app bundle / DMG.
- Windows x64 NSIS installer and portable zip.
- Bundled modern GM OpenSSH engine and legacy EC-GM compatibility engine.

## Known Limitations

- Windows terminal backend currently uses a Qt `QProcess` pipe mode, not a full ConPTY backend. Normal SSH shell use is supported, but full-screen TUI programs may need a future ConPTY implementation.
- Legacy GM adaptation reduces host-signature verification strength and should only be enabled for old EC-GM servers that require it.
