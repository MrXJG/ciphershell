# CipherShell v0.1.1

发布类型：补丁版本（patch）。

## 变更范围

- 修复终端工作区字号缩放稳定性问题。
- 统一两种渲染终端的缩放行为：
  - 文本终端（`QPlainTextEdit`）
  - 内嵌 Web 终端（`xterm.js` + Qt WebEngine）
- 同步版本与打包基线到 `0.1.1`。

## 用户可见变化

- 修复“多次缩放后偶发只能放大、无法缩小”的问题。
- `Cmd/Ctrl + 0` 现在会重置为当前会话真实初始字号。
- 统一缩放快捷键行为：
  - `Cmd/Ctrl + =` 放大
  - `Cmd/Ctrl + -` 缩小
  - `Cmd/Ctrl + 0` 重置
  - `Cmd/Ctrl + 鼠标滚轮` 缩放
