# CipherShell v0.1.0

Release type: first public release.

## Scope

- Product name: `CipherShell`.
- Terminal renderer: embedded Web terminal (Qt WebEngine + WebChannel) on macOS and Windows.
- Built-in GM dual-engine bundle:
  - modern engine: `ssh` / `sftp`
  - legacy compatibility engine: `ssh-legacy-ecgm` / `sftp-legacy-ecgm`
- Unified session/audit workflow:
  - encrypted local credential storage
  - audit log output and in-app audit view
  - SFTP and terminal session orchestration
