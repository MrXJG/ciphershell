# CipherShell v0.1.1

Release type: patch release.

## Scope

- Terminal workspace font zoom reliability fix.
- Unified zoom behavior across renderers:
  - text terminal (`QPlainTextEdit`)
  - embedded web terminal (`xterm.js` via Qt WebEngine)
- Version bump and packaging baseline refresh to `0.1.1`.

## User-visible Changes

- Fixed intermittent issue where repeated zoom operations could get stuck in an "only zoom in" state.
- `Cmd/Ctrl + 0` now resets to the actual session initial font size.
- Zoom shortcuts now behave consistently:
  - `Cmd/Ctrl + =` zoom in
  - `Cmd/Ctrl + -` zoom out
  - `Cmd/Ctrl + 0` reset
  - `Cmd/Ctrl + Mouse Wheel` zoom
