# Changelog

  - added `Cmd/Ctrl + wheel` zoom behavior with bounded font range and stable reset baseline
- Packaging/version update:
  - bumped app version to `0.1.1`
  - updated macOS/Windows packaging scripts to emit `0.1.1` artifacts
